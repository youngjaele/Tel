# 📗 개인 과제
자유 주제

## 📄 프로젝트 소개
2D 횡스크롤 게임

## 🕐 개발 기간
2023년 12월 26일 - 2023년 12월 29일

### 👪 맴버
- 개인과제
  
### 💻 개발 환경
- `Unity` `Editor Version 2022.3.2f1`
- `c#` Visual Studio 2022

### ❕ 요구사항
- 필수 요구사항
- 1. 인트로 씬 구성
    - UI 구성
    - 시작버튼 추가
    - 시작 버튼시 씬 전환 - 메인 게임으로 이동
- 2. 자유 게임 만들기
    - 전환 된 씬에서는 본인이 만들어보고 싶은 게임을 시작
    - 내용의 제한은 없습니다.
    - 진짜 어떤 게임을 만들지 도저히 생각이 안난다면 보세요.
      
### ❕ 기능
- StartScene => MainScene 전환
- W, A, S, D 이동
- Spacebar 점프 ( 더블점프 )
- Mouse LeftButton 공격
- Esc 메뉴 ( 돌아가기, 게임종료 )
